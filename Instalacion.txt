Comandos de Instalación de Librerías (NPM)
Debes ejecutar estos comandos dentro de la carpeta de tu proyecto para que el archivo package.json se actualice y las carpetas de código se descarguen:

# 1. Instalar la librería de conexión con la base de datos (Supabase)
npm install @supabase/supabase-js

# 2. Instalar la librería para el Dashboard y Gráficas de barras
npm install recharts

# 3. Instalar las librerías para generar reportes en PDF y tablas automáticas
npm install jspdf jspdf-autotable

# 4. Instalar el framework de diseño y estilos (Tailwind CSS)
npm install -D tailwindcss postcss autoprefixer

# 5. Inicializar la configuración de Tailwind (Crea los archivos de control de diseño)
npx tailwindcss init -p

Resumen de Aplicativos a Instalar (Programas de Windows)
Para que esos comandos funcionen en tu sistema, recuerda que debes haber instalado previamente estos tres programas:

Node.js (Versión LTS): Es el más importante; sin él, el comando npm no existe.  
Git for Windows: Necesario para descargar algunas librerías y para subir tu código a Netlify o GitHub.  
Visual Studio Code: El editor recomendado para manejar la terminal de comandos de forma integrada.
¿Cómo verificar que todo está instalado?

Si alguna vez tienes dudas de si te falta alguna librería, puedes abrir tu archivo package.json en la raíz del proyecto. Ahí deberías ver una lista llamada "dependencies" donde aparecerán supabase, recharts, jspdf, entre otros.
Si el archivo existe pero la aplicación no arranca, simplemente ejecuta:
